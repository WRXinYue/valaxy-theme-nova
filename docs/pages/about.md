title: 关于
---

Nova 是 [Valaxy](https://valaxy.site/) 的一个文档主题，由 [WRXinYue](https://github.com/WRXinYue) 维护。主题曾用名为 **Oceanus**，现已更名为 **Nova**（npm 包为 `valaxy-theme-nova`）。Valaxy 是一款静态站点生成器（SSG），专为构建快速且内容驱动的博客网站而设计。并且支持 Vue 3 和 Vite。简而言之，Valaxy 通过获取用 Markdown 编写的源内容，应用相应的主题，并生成可以轻松部署在任何地方的静态 HTML 页面。
